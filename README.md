# Finite-Stabilization-Theorem
```Let F be an update operator on a state space S. Under the stated monotonicity, boundedness, or finiteness conditions, the iterated sequence X(t+1)=F(X(t)) stabilizes after finitely many steps. That is, there exists N&lt;infinity such that X(N)=X(N+1), and therefore X(t)=X(N) for all t>=N.
```
